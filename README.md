# LD-19 driver

# How to use

1. Clone this repository and change `Serial(port='/dev/tty.usbserial-0001'...)` in main.py to your own port.
2. Run `pip install -r requirements.txt` in venv environment.
3. Run `python main.py`.
4. Press the E key to exit.
